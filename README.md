# game_retention_analysis

## 🏠프로젝트 제목 
모바일 게임 유저 행동 로그 분석 프로젝트


## 📌프로젝트 개요
Kaggel의 Player Analytics Event Stream Data를 활용하여</br>
모바일 게임 유저의 실시간 행동 데이터 분석을 진행하였다.

2025-02-28 ~ 2025-03-04 기간의 클래식 리텐션 분석 결과,</br> 
신규 유저의 이탈이 D0~D1 구간에서 가장 크게 발생하는 패턴을 확인하였다.

이에 따라 '초기 행동이 리텐션에 미치는 영향'을 중심으로</br>
문제 정의 → 가설 설정 → 행동 지표 설계 → 세그먼트 분석 → 전략 도출의 구조로 분석을 진행하였다.


## 🎯문제 정의 
신규 유저의 리텐션은 가입 직후 급격히 감소하며,</br>
특히 D0~D1 구간에서 가장 큰 이탈이 발생한다.

👉 "유저의 리텐션은 가입 첫날 어떤 행동에 의해 결정되는가?"


## 💡가설
1. 첫날 광고 노출이 증가할수록 리텐션은 증가하는가?
2. 첫날 성장/보상 관련 이벤트 참여도가 높을 수록 리텐션은 증가하는가?
3. 첫날 구매 유저는 비구매 유저와 다른 행동 패턴을 보이는가?


## 📊주요 분석 결과
모든 분석은 가입 첫날(D0)을 기준으로 수행하였다.</br>

1. 광고 노출 수, 성장/보상 이벤트 수, 이벤트 다양성 모두 리텐션과 양의 상관관계를 보였다.
2. 광고 노출 수와 이벤트 참여를 교차 분석한 결과, 단순 노출보다 '행동 참여(몰입도)'가 리텐션에 더 큰 영향을 미치는 것으로 나타났다. 
3. event_proxy_cnt(행동량)를 기준으로 유저를 세 구간으로 분류한 결과, 몰입 구간에 따라 리텐션이 크게 차이났다.
4. 구매 유저는 비구매 유저 대비 평균적으로(중앙값 참조) 더 높은 행동량과 다양한 이벤트 참여를 보였으나, 리텐션 상승과의 직접적인 관계는 제한적으로 나타났다.
5. 구매는 가입 직후 단시간 내에 집중되는 경향을 보였으며, 이는 구매가 몰입의 결과라기보다 초기 성향 또는 특정 행동 패턴에 의해 발생할 가능성을 시사한다. 


## 🧠인사이트
유저의 리텐션은 단순 행동량이 아니라 '몰입 구간'에 의해 결정디는 경향을 보였다.

event_proxy_cnt 기준으로 유저를 구간화한 결과:
- 저몰입 (≤30): 14.3%
- 중몰입 (31~100): 18.1%
- 고몰입 (≥101): 31.2%

중몰입 → 고몰입 전환 시 리텐션이 약 13%p 상승하였다.</br>
이는 리텐션 개선을 위해서는 단순한 행동량 증가가 아니라,</br> 
몰입 구간 전환 수준의 '의미 있는 행동 참여 증가'가 중요함을 의미한다.
